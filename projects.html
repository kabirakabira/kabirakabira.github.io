06/2023

Demographics in Reach (DiR)

For some projects, Census geographies (counties, tracts, etc) might not be optimal. This application takes a user-inputted point (XY coordinates) and pulls a set of demographic data for the area within a specified walk or drive time. This application uses the OSRM library to create an isochrone, then intersects the isochrone with Census tracts and pulls and summarizes demographic data for the area covered by the isochrone. The data can be downloaded as a geopackage.
