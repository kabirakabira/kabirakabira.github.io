Latest Project

Classifying Opportunity Zones using ML Algorithms

I tested the capability of logistic, knn and random forest classification models to be able to predict a census tract's designation as a Qualified Opportunity Zone based on 2015 American Community Survey data. Spoiler: The models didn't do too well, but it was a fun learning experience. It also revealed that not all States followed similar guidelines when nominating Census Tracts for QOZ designation.
